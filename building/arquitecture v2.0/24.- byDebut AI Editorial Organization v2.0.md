# Communication Protocol

Agents should never communicate directly through prompts.

They communicate through structured contracts.

Each message contains:

* Workflow ID
* Story Passport ID
* Story Lineage ID (if applicable)
* Input payload
* Output payload
* Confidence
* Reasoning summary
* Next recommended action

This makes the organization observable and auditable.

# Model Independence

This is one of the most important architectural principles.

The AI Editorial Organization must never depend on a single model provider.

Instead, the organization should route work to the best available model for each task.

For example:

* Deep reasoning
* Fast summarization
* Translation
* Vision
* Embeddings
* Safety

The orchestration layer owns the workflow; models are interchangeable execution engines.

This keeps byDebut resilient as the AI landscape evolves.
