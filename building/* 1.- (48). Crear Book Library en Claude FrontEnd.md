## 48. Crear Book Library en Claude FrontEnd

Indícale a Claude que construya la pantalla:

```text
Book Library
```

Debe leer desde Supabase:

```ts
const { data, error } = await supabase
  .from("books")
  .select("*")
  .order("created_at", { ascending: false });
```

Debe mostrar cada libro como card con:

```text
cover_url
title
subtitle
author_name
genre
language
status
visibility
```

Acciones visibles:

```text
Open Book
Start Transformation
Create Embedding
```

## 49. Crear Chapter Reader

Claude debe crear una pantalla donde, al abrir un libro, consulte:

```ts
const { data, error } = await supabase
  .from("chapters")
  .select("*")
  .eq("book_id", bookId)
  .order("chapter_number", { ascending: true });
```

Debe mostrar:

```text
chapter_number
title
summary
content
```

Botón principal:

```text
Transform this chapter
```

const { data, error } = await supabase
