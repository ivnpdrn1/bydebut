## 51. Crear Generate Version Screen

Claude debe crear una pantalla donde el usuario vea:

```text
Libro seleccionado
Capítulo seleccionado
Variables seleccionadas
Prompt extra
Botón Generate Version
```

Debe llamar la Edge Function:

```ts
const { data, error } = await supabase.functions.invoke("generate-version", {
  body: {
    user_id: user.id,
    book_id: selectedBook.id,
    chapter_id: selectedChapter.id,
    generation_mode: "literary",
    variables: selectedVariables,
    prompt_extra: promptExtra,
    visibility: "private"
  }
});
```
